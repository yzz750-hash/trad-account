# Trad Account — Technical Specification & Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the intelligent foreign trade financial system (智能外贸财务系统) — AI-powered voucher generation, multi-ledger, period closing, OEM commission, print/export.

**Architecture:** Monorepo — Next.js 16 (React 19, TailwindCSS 4) frontend + Python FastAPI backend. PostgreSQL for production, SQLite for dev/test. Redis sliding-window rate limiter with in-memory fallback. LiteLLM for multi-model AI routing. OpenDataLoader-PDF for OCR.

**Tech Stack:** Next.js 16.2, React 19.2, TailwindCSS 4, TypeScript 5 · Python 3.14, FastAPI, SQLAlchemy, Alembic · PostgreSQL/SQLite · Redis (optional)

## Current State Baseline

- **Tests:** 222/222 passing
- **Routers:** 14/14 (auth, users, accounts, ledgers, partners, vouchers, reports, closing, tax, system, audit, backup, export, ai_chat)
