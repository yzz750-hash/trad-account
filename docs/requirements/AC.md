## 模块八：标准打印与导出模块
### AC-19: 会计凭证打印排版
- **Given** 用户在凭证详情页点击“打印”按钮。
- **Then** 系统调用浏览器原生的打印功能。
- **And** 凭证严格遵循标准财务格式居中显示，包含：凭证字号、日期、附件数、摘要、科目、借贷金额（大写+小写金额网格）、制单/复核人签名区。
- **And** 通过专门的 `@media print` CSS 控制，完美支持标准 A4 (一页两张)、A5 (一页一张) 以及精准定制的 **宽 24cm、高 14cm** 连续打印纸规格，保证网格不跑偏。
### AC-20: 报表导出与打印
- **Given** 用户查看当月“利润表”或“OEM 订单销售报告”。
- **When** 用户点击“导出 Excel”或“打印”。
- **Then** 导出的 Excel 格式规整，打印出的纸质报表自动适配页宽。
