# 核心功能 User Stories (用户故事)

**模块一：智能凭证与平账模块**
## 史诗故事 1 (Epic 1)
作为财务人员，我希望上传原始凭证后，AI 自动提取信息、生成凭证并处理各类往来款项的复杂对账。
### US-01至US-04: (发票/流水解析、智能凭证、全往来款自动平账)

**模块八：标准打印与导出模块 (Print & Export)**
## 史诗故事 8 (Epic 8)
作为财务人员，由于审计和归档的要求，我希望系统内的任何凭证、报表、账册都能够直接生成标准尺寸的打印件或导出。
### US-19: 会计凭证打印
**作为** 财务人员，**我希望** 在查看凭证时可以一键点击“打印”，系统能够渲染出符合国家标准格式的会计凭证，支持 A4、A5 以及特定的 24cm x 14cm 宽幅格式打印纸。
### US-20: 报表与账册打印/导出
**作为** 财务人员，**我希望** 能将资产负债表、利润表、明细账册直接打印出纸质版，或者一键导出为 Excel 文件。
